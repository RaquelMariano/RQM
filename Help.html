About the Responsive Building Envelope Solutions (RBES) Database Application

This application was developed in the context of a dissertation for obtaining the Master of Science Degree in Civil Engineering in Instituto Superior Técnico. The work focuses on responsive solutions applied in envelopes of buildings all over the world. A RBES can be defined as a solution applied in the envelope of a building which displays a dynamic response to indoor and/or outdoor conditions and/or to the occupants preferences, which are implemented with the main goal of improving indoor comfort and saving energy in buildings. The dissertation can be consulted for deeper understanding of the present database and referenced as:
Mariano, R., 2018, Responsive Building Envelope Solutions, Master Thesis, Técnico Lisboa, Lisboa, Portugal

How to use this application

To view the complete tables that constitute the database simply click on the links below the "Database Tables" title. Each table holds information about a specific entity. -> Building Table - Holds information regarding every building, namely the building's name, a picture, location, designers, and more. -> Application Table - Holds information regarding every RBES application on a given building, namely where in the building's envelope the RBES is located, its orientation, control type, mechanism, and more. -> Solutions Table - Holds a short description of each type of responsive solution. -> Double-Skin Façades Table - Holds information regarding characterization of every double-skin façade application, namely the ventilation type, ventilation mode, cavity partition, input, output and more. -> Openings Table - Holds information regarding characterization of every opening application, namely the type, input, output, user override and more. -> Shading Systems Table - Holds information regarding characterization of every shading system application, namely type, opacity, input and more. -> Kinetic Envelopes Table - Holds information regarding characterization of every kinetic envelope application, namely scale, motion, input and more. -> Smart Materials Table - Holds information regarding characterization of every smart material application, namely type, subtype, input and more. -> Green Systems Table - Holds information regarding characterization of every green system application, namely type and plants. In order to demonstrate that the information in these tables can be easily filtered and researched in ways that are useful to the information consumers, some possible interactions were programmed.
