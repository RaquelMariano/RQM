About the Responsive Building Envelope Solutions (RBES) Database Application

This application was developed in the context of a dissertation for obtaining the Master of Science Degree in Civil Engineering in Instituto Superior Técnico. The work focuses on responsive solutions applied in envelopes of buildings all over the world. A RBES can be defined as a solution applied in the envelope of a building which displays a dynamic response to indoor and/or outdoor conditions and/or to the occupants preferences, which are implemented with the main goal of improving indoor comfort and saving energy in buildings. The dissertation can be consulted for deeper understanding of the present database and referenced as:
Mariano, R., 2018, Responsive Building Envelope Solutions, Master Thesis, Técnico Lisboa, Lisboa, Portugal

How to use this application

To view the complete tables that constitute the database simply click on the links below the "Database Tables" title.
